# 🎬 Movie Booking System

A simple full-stack movie booking web application built using Spring Boot and JavaScript.

## 🚀 Features
- View showtimes
- Select seats (visual UI)
- Book tickets
- Payment page simulation
- Booking history
- Cancel booking
- Seat availability tracking

## 🛠️ Tech Stack
- Backend: Spring Boot (Java)
- Frontend: HTML, CSS, JavaScript
- Storage: In-memory + LocalStorage

## ▶️ How to Run
1. Clone the repo
2. Run Spring Boot application
3. Open: http://localhost:8080/index.html
